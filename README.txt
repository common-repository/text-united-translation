== Installation ==

1. Add the Translate WordPress Websites with TextUnited plugin to your WordPress dashboard and activate it.
2. Navigate to the TextUnited plugin and follow the onscreen instructions to create an account. This will automatically connect the plugin to our systems.
3. Alternatively, if you already have a TextUnited account, you can log in to your [TextUnited dashboard](https://www.textunited.com/my/login/), navigate to the API menu, and copy your Company ID and API Key to connect the plugin to our system.
4. You’re ready to start! If you are having problems, refer to this [helpdesk article](https://help.textunited.com/how-to-translate-your-website-using-the-text-united-wordpress-plugin) to guide you through connecting the plugin.

== Frequently Asked Questions ==

= Is Translate WordPress Websites with TextUnited free? =
The Translate WordPress Websites with TextUnited plugin is free to download and a free account with TextUnited lets you translate 1000 new words per month totally free, in any language.

Our WordPress multilingual plugin makes translation fast and affordable, and users can either buy word packages to top up their free account or check out our subscriptions to access the full TextUnited translation management system

TextUnited also offers a 14-day free trial version; try our multilingual website solution today!.

= Is Translate WordPress Websites with TextUnited compatible with SEO?  =
Translate WordPress Websites with TextUnited is SEO optimized for multilingual websites. Meaning search engines like Google will index translated pages that contain your multilingual content.

= Can I migrate from WPML, Polylang or Weglot to Translate WordPress Websites with TextUnited? =
Yes, you can easily switch from Weglot, Polylang, WPML or any other WordPress translation plugin. Simply deactivate your existing multilingual translation plugin, and you’ll be able to use Translate WordPress Websites with TextUnited! If you want to import your existing translation memory from another provider’s translation plugin, get in touch with us at support@textunited.com and we can talk you through the process!

= How do I set up Translate WordPress Websites with TextUnited? =
We’ve made setting up Translate WordPress Websites with TextUnited as simple as can be - we’ve even got a video to help guide you through it

[youtube https://www.youtube.com/watch?v=xO5kkBPw_IU]

You can also check out our [Helpdesk article](https://help.textunited.com/translating-your-wordpress-site-with-the-textunited-plugin) complete with a step-by-step guide and screenshots.

= Can I edit my translations? =
You can use the Overlay Editor to edit your translations in context so you can see exactly how it will look before you update it!

= Can Translate WordPress Websites with TextUnited translate URLs? =
Yes, you can translate your URLs with TextUnited. When you connect the plugin and you add pages, the URL package is created. You can then log in to your TextUnited Portal, open the task, and translate the slugs.

= Can I use Translate WordPress Websites with TextUnited on more than one website? =
Yes! On TextUnited you can create as many translation projects as you wish, therefore you can use WordPress to translate many websites.

= Does Translate WordPress Websites with TextUnited provide support? =
Yes! You can contact us for support via email at support@textunited.com or by starting a chat conversation on our website

= How many languages does Translate WordPress Websites with TextUnited support? =
The Translate WordPress Websites with TextUnited plugin offers support for up to 170 languages!

= Where are my website translations stored? =
The translations are stored in your TextUnited account and buffered in your WordPress. This means they are available to visitors to your website directly from WordPress.

= Does TextUnited WordPress plugin provide the server-side translation? =
Yes, translated pages are prepared by your WordPress and they are delivered to the visitor’s browser already translated.
